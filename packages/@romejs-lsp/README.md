# Language Server Protocol

## VSCode

### Prerequisites

- Install VSCode
- Setup terminal path

```
code ./packages/@romejs-lsp/vscode
```

## Atom

### Prerequisites

- Install Atom

```
cd ./packages/@romejs-lsp/vscode
npm run compile
cd ../atom
apm link 
```

```
atom .
```

If you already have Atom open then open the command pallette and run `reload`
